JSON (JavaScript Object Notation) : 자바스크립트에서 제공하는 객체를 표현하는 방법 => 표준데이터 포멧 (XML,CSV) * 장점 : - 단순 텍스트 형태로 표기 (직관적-이해도 증가) - 속성(key), 값(value) 형태의 쌍으로 이루어진 데이터 - 필요한 데이터의 크기가 상대적으로 적음 (부산, 'itwill':'부산') - 특정언어(javascript)에 독립적이다. => 문법에 민감하다 - 최근에 사용된 대부분의 브라우저에서 기본적으로 제공 - 다른 도메인주소에서도 요청을 처리할 수 있음 * 단점 : - 문법에 민감 (제공되는 데이터타입만 사용가능) - 악성코드(자바스크립트)를 포함가능 -> 신뢰가능한 json데이터 활용 해야함 속성(key), 값(value) 쌍으로 구성 -> "속성" : 값, "속성" : 값 * 정수 데이터 (2진수,8진수,16진수 지원x) * 실수 데이터 (고정, 부동소수점) 3.124 -1.232 / 1e4 6.3E-5 * Boolean true false * Null * 문자열 ("" 사용하여 0개이상의 유니코드로 구성, 이스케이프문법) * JSON 객체 {"속성":값 , ....} * JSON 배열 [ 10,"aa",{"속성":값} ] ** Date 시간,날짜 정보 사용불가!!! -> String 변경 "2023-02-22" json simple
